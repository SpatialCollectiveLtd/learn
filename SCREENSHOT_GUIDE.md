## üìê SCREENSHOT SPECIFICATIONS

### Technical Requirements:
- **Format**: PNG
- **Resolution**: 1920x1080 minimum (Full HD)
- **Compression**: Medium (balance quality and file size)
- **File Size**: Under 500KB per image (optimize if larger)

### Capture Guidelines:
1. **Clean Interface**: Close unnecessary windows/tabs
2. **Focus**: Highlight the specific feature being demonstrated
3. **Context**: Include enough surrounding UI for orientation
4. **Annotations**: Add red arrows/circles where needed
5. **Privacy**: Blur any personal information (emails, usernames)

### Naming Convention:
`[number]-[description].png`
- Example: `01-gmail-signup.png`
- Sequential numbering (01-42)
- Lowercase with hyphens
